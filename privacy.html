Nexora Privacy Policy

Last Updated: July 10, 2026

Nexora respects your privacy. This Privacy Policy explains what information we collect, how we use it, and how we protect it.

Information We Collect

  • Discord Server ID
  • Discord User ID
  • Channel ID
  • Role ID
  • Configuration settings created by server administrators

Nexora does not collect passwords or private Discord messages unless a feature explicitly requires message processing (such as moderation or AI commands).

How We Use Information

  • Provide moderation features
  • Store server settings
  • Operate ticket systems
  • Generate analytics
  • Provide AI features
  • Improve the service

Data Security

We take reasonable measures to protect stored information from unauthorized access.

Third-Party Services

Nexora may use third-party services such as Discord and AI providers (including Groq) to provide certain features.

Your Rights

Server administrators may request deletion of stored server configuration data by contacting Nexora Support.

Changes

This Privacy Policy may be updated at any time. Continued use of Nexora means you accept the updated policy.

Contact

If you have questions, please contact the Nexora Support Team.
